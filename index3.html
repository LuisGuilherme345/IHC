C++ é uma linguagem de programação de alto nível, de uso geral, que oferece controle sobre o hardware, sendo uma extensão da linguagem C. É conhecida pela sua eficiência e versatilidade, sendo usada em diversas aplicações, desde sistemas operacionais e jogos até desenvolvimento web e software de baixo nível.

O Java é uma linguagem de programação popular orientada a objetos e multiplataforma. O Java pode ser usado como uma plataforma por meio das máquinas virtuais Java (JVMs), que podem ser instaladas na maioria dos computadores e dispositivos móveis.

HTML (HyperText Markup Language) é a linguagem de marcação fundamental para a criação de páginas web. Serve para estruturar o conteúdo de um site, definindo elementos como textos, imagens, links e outros componentes que são depois apresentados nos navegadores.

CSS (Cascading Style Sheets) é a linguagem usada para definir a aparência visual de uma página web, separando a estrutura (HTML) da apresentação. É responsável por controlar elementos como cores, fontes, espaçamento, layout e efeitos de animação, tornando possível criar páginas web visualmente atraentes e bem organizadas.
